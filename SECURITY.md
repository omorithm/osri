# Security Policy

## Overview

The security and trustworthiness of OSRI are important to the project.

OSRI is designed to analyze open-source software projects using publicly available signals. We are committed to maintaining a secure, transparent, and responsible development process.

This document explains how to report potential security vulnerabilities in OSRI.

---

## Supported Versions

As OSRI is currently under active development, security fixes will generally be applied to the latest available version.

| Version | Supported |
| ------- | ---------- |
| Latest release | ✅ |
| Development branch | ✅ |
| Older versions | ❌ |

---

## Reporting a Vulnerability

If you discover a potential security vulnerability in OSRI, please report it responsibly.

Please do not publicly disclose security issues through GitHub Issues, discussions, or pull requests before they have been reviewed.

When reporting a vulnerability, include:

- A description of the issue
- Steps to reproduce the problem
- Potential impact
- Relevant logs, screenshots, or examples where applicable
- Any suggested mitigation

---

## Response Process

After receiving a security report, maintainers will:

1. Review the reported issue
2. Confirm whether it is a valid vulnerability
3. Investigate potential impact
4. Develop and test a fix where necessary
5. Communicate updates with the reporter

Response times may vary depending on the complexity and severity of the issue.

---

## Responsible Disclosure

OSRI encourages responsible disclosure.

Security researchers and contributors are encouraged to provide sufficient time for vulnerabilities to be investigated and addressed before public disclosure.

We appreciate the efforts of individuals who help improve the security and reliability of the project.

---

## Security Scope

Security concerns may include:

- Vulnerabilities in OSRI code
- Dependency vulnerabilities
- Data handling issues
- Authentication or authorization issues (where applicable)
- Issues that could compromise users of OSRI

---

## Thank You

Thank you for helping keep OSRI secure and trustworthy.
